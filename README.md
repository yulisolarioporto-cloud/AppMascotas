##
**AppMascotas** es una aplicación móvil desarrollada en Android que permite gestionar información de mascotas mediante un sistema CRUD (Crear, Leer, Actualizar y Eliminar), conectándose a una API para el manejo de datos.

La aplicación facilita el registro y control de mascotas, permitiendo:
- Visualizar mascotas
- Agregar nuevas mascotas
- Editar información existente
- Eliminar mascotas de manera sencilla

---------------------------
##Operaciones soportadas por la API REST

| Método | Acción                     |
|--------|--------------------------- |
| GET    | Obtener todas las mascotas |
| POST   | Registrar una nueva mascota|
| PUT    | Actualizar información     |
| DELETE | Eliminar una mascota       |

Requisitos

- Android Studio (última versión recomendada)  
- Dispositivo o emulador Android  
- Conexión a Internet
